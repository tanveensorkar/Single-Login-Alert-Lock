# Single-Login-Alert-Lock

A lightweight WordPress plugin designed to improve security, user session management, and account access control.

## Requirements

- WordPress 6.0+
- PHP 7.4+
- Administrator access

## Installation

1. Download the plugin ZIP file.
2. Go to WordPress Admin → Plugins → Add New.
3. Click Upload Plugin.
4. Select the ZIP file and install.
5. Activate the plugin.
6. Configure settings from the WordPress dashboard.

Short Description

Prevent multiple device logins by allowing only one active session per user and displaying a warning when another login attempt is detected.

Full Description

Single Login Alert Lock enhances account security by ensuring that each user can only be logged in from one device at a time. When a second login attempt is made, the plugin blocks access and displays a clear warning message instead of disconnecting the current session.

Features
One active login per user
Detects simultaneous login attempts
Displays warning messages for blocked logins
Improves account protection
Lightweight and easy to use
Ideal For
Membership websites
Online courses
Subscription-based platforms
Client portals
