# Single Login Alert Lock

Prevent multiple device logins by allowing only one active session per user and displaying a warning when another login attempt is detected.

---

## 📖 Overview

**Single Login Alert Lock** is a lightweight WordPress security plugin designed to prevent simultaneous logins across multiple devices or browsers.

Unlike traditional single-session plugins that automatically terminate existing sessions, this plugin keeps the currently active session intact and blocks any new login attempts. Users attempting to log in from another device will receive a clear warning message, helping protect accounts from unauthorized access and account sharing.

---

## ✨ Features

* Enforce one active login session per user
* Detect simultaneous login attempts
* Block new login attempts when a session is already active
* Display customizable warning messages
* Protect user accounts from unauthorized access
* Prevent account sharing
* Lightweight and performance-friendly
* Easy setup and configuration

---

## 🔒 How It Works

1. A user logs into your WordPress website.
2. The session becomes active and protected.
3. If the same account attempts to log in from another device or browser:

   * The existing session remains active.
   * The new login attempt is blocked.
   * A warning message is displayed to the user.
4. Only the original session retains access.

## 🚀 Installation

1. Download the plugin ZIP file.
2. Navigate to **WordPress Admin → Plugins → Add New**.
3. Click **Upload Plugin**.
4. Select the ZIP file and install.
5. Activate the plugin.
6. Configure the plugin settings from the WordPress dashboard.
